Steps to run the project - 
1. Open the terminal and go to /src folder
2. Execute the command 'dotnet run' and wait for the project to build and start to run
3. Open the another terminal/terminals to simulate the client in the project folder.
4. Execute the command 'dotnet fsi "--langversion:preview" .\client.fsx'
5. The clinet process will start running, and you will be presented with the options to use the twitter server.
